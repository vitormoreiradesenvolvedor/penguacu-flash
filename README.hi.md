# Penguaçu-Flash

### Linux से Windows 7 · 8.1 · 10 · 11 के बूटेबल USB बनाएँ — आसान तरीके से

एक पूरी तरह स्वयं-निहित AppImage जो आपकी USB को फ़ॉर्मेट करता है, Windows कॉपी करता है और एक वैयक्तिकृत `autounattend.xml` इंजेक्ट करता है — ताकि इंस्टॉलेशन बिना किसी हस्तक्षेप के चले, आपके खाते, भाषा और गोपनीयता सेटिंग्स पहले से तय।

## ✨ Penguaçu-Flash क्यों?

Linux पर Windows इंस्टॉल USB बनाने का मतलब आमतौर पर `parted`, `mkntfs`, `7z` और बूट-सेक्टर टूल्स को हाथ से चलाना होता है — और फिर भी Windows इंस्टॉल के बीच में सवाल पूछकर रुक जाता है। Penguaçu-Flash यह सब एक ही विंडो में करता है:

- 🐧 **कहीं भी चले** — एक स्वयं-निहित AppImage। कोई निर्भरता इंस्टॉल नहीं करनी: `7-Zip`, `mkntfs` और `ms-sys` अंदर ही शामिल हैं।
- 💾 **असली बूटेबल USB** — पार्टिशन करता है, NTFS फ़ॉर्मेट करता है, Windows कॉपी करता है और **UEFI तथा BIOS/लीगेसी** दोनों के बूट सेक्टर लिखता है।
- 🪄 **वास्तव में अनअटेंडेड** — एक वैयक्तिकृत `autounattend.xml` इंजेक्ट करता है ताकि सेटअप सवालों को पार कर जाए, उपयोगकर्ता नाम, पासवर्ड, कंप्यूटर नाम, समय क्षेत्र और संस्करण पहले से भरे हुए।
- 🧭 **Windows 7 → 11, स्वतः पहचान** — संस्करण ISO फ़ाइल नाम से पहचाना जाता है, और उत्तर फ़ाइल हर संस्करण के स्कीमा के अनुरूप ढल जाती है।
- 🛡️ **बिना रुकावट Windows 11** — TPM 2.0, Secure Boot, RAM और CPU आवश्यकताओं का वैकल्पिक बायपास।
- 🔒 **शुरुआत से गोपनीयता** — टेलीमेट्री, Cortana, स्वचालित BitLocker, प्रायोजित ऐप्स आदि बंद करें।
- 📊 **ईमानदार प्रगति** — प्रगति बार *डिवाइस पर भौतिक रूप से लिखे गए बाइट्स* को ट्रैक करती है (लाइव MB/s के साथ), न कि भ्रामक RAM-कैश संख्या को।
- 🌐 **6 भाषाएँ बोलता है** — पुर्तगाली, अंग्रेज़ी, स्पेनिश, चीनी, हिन्दी और अरबी (पूर्ण दाएँ-से-बाएँ लेआउट के साथ)।

## 🚀 डाउनलोड करें और चलाएँ

1. [**Releases**](../../releases) पेज से नवीनतम **`Penguaçu-Flash-*.AppImage`** प्राप्त करें।
2. इसे निष्पादन-योग्य बनाएँ और चलाएँ:

```bash
chmod +x Penguaçu-Flash-*.AppImage
./Penguaçu-Flash-*.AppImage
```

बस इतना ही — कोई इंस्टॉलेशन नहीं, कोई निर्भरता नहीं। विशेषाधिकार वाले चरण (पार्टिशन, फ़ॉर्मेट, बूट सेक्टर लिखना) सिस्टम के मानक प्राधिकरण संवाद के माध्यम से एक बार आपका पासवर्ड माँगते हैं।

> **सुझाव:** KDE/GNOME पर आप फ़ाइल प्रबंधक में AppImage पर बस डबल-क्लिक भी कर सकते हैं।

## 🧭 उपयोग कैसे करें

| चरण | क्या होता है |
|-----|-------------|
| **1. ISO** | अपनी Windows ISO डालें। संस्करण (7/8.1/10/11) स्वतः पहचाना जाता है। |
| **2. पहचान** | उपयोगकर्ता नाम, पासवर्ड, कंप्यूटर नाम, समय क्षेत्र, संस्करण और (वैकल्पिक) उत्पाद कुंजी सेट करें। |
| **3. सेटिंग्स** | गोपनीयता/व्यवहार सेटिंग्स चुनें और Windows 11 के लिए आवश्यकता बायपास। |
| **4. बनाएँ** | **ISO बनाएँ** (एक नई कस्टम `.iso`) या **USB बनाएँ** (फ़ॉर्मेट + कॉपी + बूट-तैयार USB इंजेक्ट) चुनें। |
| **5. पूर्ण** | लक्ष्य मशीन को USB से बूट करें — Windows आपकी सेटिंग्स के साथ स्वयं इंस्टॉल होता है। |

आप **मौजूदा USB का विश्लेषण** भी कर सकते हैं, उसकी `autounattend.xml` पढ़ सकते हैं और बिना कुछ दोबारा बनाए अद्यतन कॉन्फ़िगरेशन फिर से इंजेक्ट कर सकते हैं।

## 🛠️ स्रोत से बिल्ड करें

आवश्यकताएँ: **Node.js 18+**, और बिल्ड मशीन पर `ntfsprogs` (`mkntfs` के लिए) तथा `gcc`/`make` (अंतर्निहित `ms-sys` कंपाइल करने के लिए)।

```bash
git clone https://github.com/vitormoreiradesenvolvedor/penguacu-flash.git
cd penguacu-flash
npm install

# अंतर्निहित बाइनरी (7-Zip, mkntfs, ms-sys) को ./bin में डाउनलोड/कंपाइल करता है
npm run prepare-bins

# dist/Penguaçu-Flash-<संस्करण>.AppImage बनाता है
npm run dist
```

विकास के दौरान चलाने के लिए:

```bash
npm start
```

## ⚠️ अस्वीकरण

बूटेबल USB बनाने से चयनित ड्राइव का **सारा डेटा मिट जाता है**। दोबारा जाँच लें कि आपने सही डिवाइस चुना है। ऐप में दिखाई गई उत्पाद कुंजियाँ Microsoft की सार्वजनिक जेनेरिक/KMS सेटअप कुंजियाँ हैं — वे इंस्टॉल के दौरान संस्करण चुनती हैं पर Windows को **सक्रिय नहीं करतीं**। वैध लाइसेंस से सक्रिय करें।
