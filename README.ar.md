# Penguaçu-Flash

### أنشئ أقراص USB إقلاعية لنظام Windows 7 · 8.1 · 10 · 11 من لينكس — بسهولة

حزمة AppImage مكتفية ذاتيًا بالكامل تقوم بتهيئة قرص USB، ونسخ Windows، وحقن ملف `autounattend.xml` مخصّص — لكي يعمل التثبيت دون تدخّل، مع حسابك ولغتك وإعدادات الخصوصية مضبوطة مسبقًا.

## ✨ لماذا Penguaçu-Flash؟

عادةً ما يتطلّب إنشاء قرص USB لتثبيت Windows على لينكس التعامل يدويًا مع `parted` و`mkntfs` و`7z` وأدوات قطاع الإقلاع — ومع ذلك يتوقّف Windows في منتصف الإعداد ليطرح أسئلة. يقوم Penguaçu-Flash بكل ذلك في نافذة واحدة:

- 🐧 **يعمل في أي مكان** — حزمة AppImage مكتفية ذاتيًا. لا حاجة لتثبيت اعتماديات: `7-Zip` و`mkntfs` و`ms-sys` مضمّنة بالداخل.
- 💾 **قرص USB إقلاعي حقيقي** — يقسّم ويهيّئ NTFS وينسخ Windows ويكتب قطاعات الإقلاع لكل من **UEFI وBIOS/التقليدي**.
- 🪄 **دون تدخّل فعليًا** — يحقن ملف `autounattend.xml` مخصّصًا ليتجاوز الإعداد الأسئلة، مع اسم المستخدم وكلمة المرور واسم الجهاز والمنطقة الزمنية والإصدار مملوءة مسبقًا.
- 🧭 **Windows 7 ← 11، كشف تلقائي** — يُتعرّف على الإصدار من اسم ملف ISO، ويتكيّف ملف الإجابة مع مخطّط كل إصدار.
- 🛡️ **Windows 11 دون العوائق** — تجاوز اختياري لمتطلّبات TPM 2.0 وSecure Boot والذاكرة والمعالج.
- 🔒 **خصوصية جاهزة** — أوقف القياس عن بُعد وCortana وBitLocker التلقائي والتطبيقات المموّلة والمزيد.
- 📊 **تقدّم صادق** — يتتبّع شريط التقدّم *البايتات المكتوبة فعليًا على الجهاز* (مع MB/s حيّة)، وليس الرقم المضلّل الخاص بذاكرة التخزين المؤقت.
- 🌐 **يتحدّث 6 لغات** — البرتغالية والإنجليزية والإسبانية والصينية والهندية والعربية (مع تخطيط كامل من اليمين إلى اليسار).

## 🚀 التنزيل والتشغيل

1. احصل على أحدث **`Penguaçu-Flash-*.AppImage`** من صفحة [**Releases**](../../releases).
2. اجعله قابلاً للتنفيذ وشغّله:

```bash
chmod +x Penguaçu-Flash-*.AppImage
./Penguaçu-Flash-*.AppImage
```

هذا كل شيء — دون تثبيت ودون اعتماديات. تطلب الخطوات ذات الصلاحيات (التقسيم، التهيئة، كتابة قطاعات الإقلاع) كلمة مرورك مرّة واحدة عبر مربّع حوار التفويض القياسي لنظامك.

> **نصيحة:** على KDE/GNOME يمكنك أيضًا النقر المزدوج على AppImage من مدير الملفات.

## 🧭 كيفية الاستخدام

| الخطوة | ما الذي يحدث |
|--------|--------------|
| **1. ISO** | أفلت ملف Windows ISO. يُكتشف الإصدار (7/8.1/10/11) تلقائيًا. |
| **2. الهوية** | اضبط اسم المستخدم وكلمة المرور واسم الجهاز والمنطقة الزمنية والإصدار و(اختياريًا) مفتاح المنتج. |
| **3. الإعدادات** | اختر إعدادات الخصوصية/السلوك، ولنظام Windows 11 تجاوز المتطلّبات. |
| **4. الإنشاء** | اختر **إنشاء ISO** (ملف `.iso` مخصّص جديد) أو **إنشاء USB** (تهيئة + نسخ + حقن قرص جاهز للإقلاع). |
| **5. تم** | أقلِع الجهاز الهدف من USB — يثبّت Windows نفسه بإعداداتك. |

يمكنك أيضًا **تحليل قرص USB موجود** وقراءة ملف `autounattend.xml` الخاص به وإعادة حقن إعداد محدّث دون إعادة بناء أي شيء.

## 🛠️ البناء من المصدر

المتطلّبات: **Node.js 18+**، وعلى جهاز البناء `ntfsprogs` (لأجل `mkntfs`) بالإضافة إلى `gcc`/`make` (لتصريف `ms-sys` المضمّن).

```bash
git clone https://github.com/vitormoreiradesenvolvedor/penguacu-flash.git
cd penguacu-flash
npm install

# يقوم بتنزيل/تصريف الملفات الثنائية المضمّنة (7-Zip، mkntfs، ms-sys) إلى ./bin
npm run prepare-bins

# ينتج dist/Penguaçu-Flash-<الإصدار>.AppImage
npm run dist
```

للتشغيل أثناء التطوير:

```bash
npm start
```

## ⚠️ إخلاء مسؤولية

إنشاء قرص USB إقلاعي **يمحو جميع البيانات** على القرص المحدّد. تحقّق مرّتين من اختيار الجهاز الصحيح. مفاتيح المنتج المعروضة في التطبيق هي مفاتيح إعداد Microsoft العامة/KMS العلنية — تختار الإصدار أثناء الإعداد لكنها **لا تُفعّل** Windows. فعّله برخصة صالحة.
